# Assignment

## Install

It requires [node.js](http://nodejs.org/download/)

* git clone https://github.com/netzwelt-rajan/assignment1.git
* cd assignment1/
* npm install
* npm start

The server will run on port 5000.
You can test it in the (Chrome or Firefox) browser at localhost:5000.
